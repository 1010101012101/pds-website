Please notify us of any problems or concerns you have with this version of the CIRS data. ### Getting the Data You Want The table below contains links to all of the available data volumes from the Saturn encounter re-formatted into fixed length data files. For information on the re-formatted data, see [About Re-formatted Data](reform_about.html) * Clicking on the VOLUME_ID will take you to the root directory of the volume. * To download an entire volume, right click on that volume's Zip link in the second column. Note: These are large files---on the order of 1 to 3 Gbytes. Downloading may take some time. Note that some volumes are too large to compress via the Zip format; these are provided in .tar.gz format instead (meaning that they are Unix tar files compressed by gnuzip).
