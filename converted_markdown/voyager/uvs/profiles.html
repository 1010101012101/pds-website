* [PPS Intro](index.html) * [Instrument](instrument.html) * [Archive Data](data.html) * [Standard Profiles](profiles.html) ![\[SPACER\]](../css/spacer.gif) ## Standard UVS Ring Profiles This page provides access to ASCII table files containing information about the rings at uniformly-spaced radial samples (0.2, 0.5, 1, 2, 5, 10, 20, 50 km). The columns tabulated are: * Radius in km * Raw PPS counts * Standard deviation in raw PPS counts * Normal optical depth * One standard deviation lower limit on optical depth * One standard deviation upper limit on optical depth (**Note**: Due to lack of calibration information, only the first three columns are tabulated for the second and third ring occultation profiles from Saturn---delta Sco ingress and iota Her egress.) Click on the data file name (*.TAB) to retrieve the data. Click on the corresponding label file (*.LBL) to retrieve the PDS label, which describes the data file contents in detail. Note that all of these files are located within subdirectories of the [EASYDATA](/vol/VG_28xx/VG_2802/EASYDATA/) directory on PDS volume [VG_2802](/vol/VG_28xx/VG_2802/). * Saturn * Uranus * Ring
