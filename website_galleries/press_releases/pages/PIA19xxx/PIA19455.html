# Hadley Crater - False Color * Click the [image above](/press_releases/medium/PIA19xxx/PIA19455_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA19455.jpg) (786 x 2252) (158.3 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA19455.tif) (786 x 2252) (5.3 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows a crater within a crater on the floor of Hadley Crater.
