# Candor Chasma - False Color * Click the [image above](/press_releases/medium/PIA19xxx/PIA19784_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA19784.jpg) (715 x 1665) (155.0 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA19784.tif) (715 x 1665) (2.6 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of the floor of Candor Chasma.
