# Noachis Terra Dunes - False Color * Click the [image above](/press_releases/medium/PIA19xxx/PIA19742_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA19742.jpg) (675 x 1441) (73.9 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA19742.tif) (675 x 1441) (1.9 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows dunes in an unnamed crater in Noachis Terra.
