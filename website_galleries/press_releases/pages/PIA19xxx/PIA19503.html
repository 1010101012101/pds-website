# Sirenum Fossae * Click the [image above](/press_releases/medium/PIA19xxx/PIA19503_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA19503.jpg) (1341 x 2650) (190.8 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA19503.tif) (1341 x 2650) (3.6 MB)

Caption:


Context image

The linear features in this VIS image are faults. Where depressions are formed bounded by faulting the feature is called a graben. These faults and graben are part of a large region of faulting called Sirenum Fossae.

Orbit Number: 59353 Latitude: -24.4596 Longitude: 220.937 Instrument: VIS Captured: 2015-05-01 20:03
