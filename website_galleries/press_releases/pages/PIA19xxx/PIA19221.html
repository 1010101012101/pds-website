# Marth Crater - False Color * Click the [image above](/press_releases/medium/PIA19xxx/PIA19221_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA19221.jpg) (687 x 1355) (58.8 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA19221.tif) (687 x 1355) (2.8 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of the floor of Marth Crater. Sand dunes are blue in this false color image.
