# Spirit's View on Sol 93 (vertical) * Click the [image above](/press_releases/medium/PIA05xxx/PIA05769_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA05769.jpg) (4000 x 4000) (1.8 MB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA05769.tif) (4000 x 4000) (14.4 MB)

Caption:

This vertical-projection mosaic was created from navigation camera images that NASA's Mars Exploration Rover Spirit acquired on sol 93 (April 7, 2004). It reveals the martian view from Spirit's position during the four-sol flight software update that began on sol 94.
