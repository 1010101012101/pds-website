# To Mars and Back * Click the [image above](/press_releases/medium/PIA05xxx/PIA05488_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA05488.jpg) (960 x 996) (116.1 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA05488.tif) (960 x 996) (2.2 MB)

Caption:

This artist's concept of the proposed Mars Sample Return mission shows the launch of the martian sample back toward Earth. This image is part of a series designed to describe the current groundbreaking Mars Sample Return mission concept (see figure 1 below for a composite of the series), with a tentative launch date of 2013.
