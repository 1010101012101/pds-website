# Spirit's View on Sol 124 (3-D) * Click the [image above](/press_releases/medium/PIA05xxx/PIA05901_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA05901.jpg) (7704 x 1250) (1.2 MB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA05901.tif) (7704 x 1250) (21.9 MB)

Caption:

This three-dimensional view in a cylindrical-perspective projection was created from navigation camera images that NASA's Mars Exploration Rover Spirit acquired on sol 124 (May 9, 2004). It reveals Spirit's view as it gets closer to the "Columbia Hills."

See PIA05902 for left eye view and PIA05903 for right eye view of this 3-D cylindrical-perspective projection.
