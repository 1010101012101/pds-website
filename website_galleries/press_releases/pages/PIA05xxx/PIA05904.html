# Spirit's View on Sol 124 (polar) * Click the [image above](/press_releases/medium/PIA05xxx/PIA05904_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA05904.jpg) (4041 x 4041) (1.3 MB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA05904.tif) (4041 x 4041) (9.1 MB)

Caption:

This polar-projection view was created from navigation camera images that NASA's Mars Exploration Rover Spirit acquired on sol 124 (May 9, 2004). It reveals Spirit's view as it gets closer to the "Columbia Hills."
