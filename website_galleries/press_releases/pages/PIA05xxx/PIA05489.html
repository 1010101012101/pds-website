# Rendezvous in Space * Click the [image above](/press_releases/medium/PIA05xxx/PIA05489_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA05489.jpg) (646 x 566) (63.4 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA05489.tif) (646 x 566) (1.0 MB)

Caption:

This artist's concept of the proposed Mars Sample Return mission shows rendezvous of the orbiting sample container with the Earth return vehicle. This image is part of a series designed to describe the current groundbreaking Mars Sample Return mission concept (see figure 1 below for a composite of the series), with a tentative launch date of 2013.
