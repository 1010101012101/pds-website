# Daedalia Planum * Click the [image above](/press_releases/medium/PIA21xxx/PIA21323_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA21323.jpg) (672 x 1419) (142.8 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA21323.tif) (672 x 1419) (752.8 kB)

Caption:


Context image

This VIS image shows a portion of Daedalia Planum, an extensive volcanic plain comprised of flows from Arsia Mons. Different flow surface textures help delineate different flows.

Orbit Number: 66856 Latitude: -18.933 Longitude: 242.941 Instrument: VIS Captured: 2017-01-08 20:07
