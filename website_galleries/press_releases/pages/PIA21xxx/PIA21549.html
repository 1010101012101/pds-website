# Proctor Crater Dunes- False Color * Click the [image above](/press_releases/medium/PIA21xxx/PIA21549_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA21549.jpg) (672 x 1437) (90.1 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA21549.tif) (672 x 1437) (2.0 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of the sand sheet with surface dune forms located on the floor of Proctor Crater. Proctor Crater is located in Noachis Terra.
