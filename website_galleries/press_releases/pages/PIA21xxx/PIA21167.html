# Candor Chasma - False Color * Click the [image above](/press_releases/medium/PIA21xxx/PIA21167_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA21167.jpg) (706 x 1546) (89.2 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA21167.tif) (706 x 1546) (2.0 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of Candor Chasma. There is a landslide deposit from the rim of the chasma.
