# Gorgonum Chaos - False Color * Click the [image above](/press_releases/medium/PIA21xxx/PIA21545_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA21545.jpg) (673 x 1432) (112.5 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA21545.tif) (673 x 1432) (2.0 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of Gorgonum Chaos. The linear depression at the bottom of the image are part of Sirenum Fossae. Gorgonum Chaos is located in Terra Sirenum.
