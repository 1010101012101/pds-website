# Cerberus Flood Features * Click the [image above](/press_releases/medium/PIA03xxx/PIA03043_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA03043.jpg) (836 x 2253) (359.7 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA03043.tif) (836 x 2253) (1.9 MB)

Caption:

16 October 2005
This Mars Global Surveyor (MGS) Mars Orbiter Camera (MOC) image shows streamlined landforms carved by catastrophic floods that occurred in the eastern Cerberus region, some time in the distant martian past.

Location near : 15.1°N, 193.5°W
