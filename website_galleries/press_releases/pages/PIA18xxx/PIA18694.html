# Dark Slope Streaks * Click the [image above](/press_releases/medium/PIA18xxx/PIA18694_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA18694.jpg) (1424 x 2943) (271.5 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA18694.tif) (1424 x 2943) (4.2 MB)

Caption:


Context image

This VIS image shows dark slope streaks in an unnamed crater in Terra Sabaea. These features are believed to be formed by material moving downslope, removing the dust cover and revealing darker material.
