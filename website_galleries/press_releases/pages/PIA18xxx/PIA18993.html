# False Color Surface * Click the [image above](/press_releases/medium/PIA18xxx/PIA18993_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA18993.jpg) (696 x 1360) (117.1 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA18993.tif) (696 x 1360) (2.8 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of the region near Nili Fossae.
