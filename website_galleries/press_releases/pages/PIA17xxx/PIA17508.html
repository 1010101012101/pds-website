# Pollack Crater * Click the [image above](/press_releases/medium/PIA17xxx/PIA17508_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA17508.jpg) (320 x 8208) (451.3 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA17508.tif) (320 x 8208) (2.6 MB)

Caption:


Context image

This IR image shows the feature called "White Rock." In vis images, the feature is brighter in color that the surroundings. In this IR image the feature is dark, indicating that it is cooler than the surroundings.
