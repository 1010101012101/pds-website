# Dark Slope Streaks * Click the [image above](/press_releases/medium/PIA17xxx/PIA17512_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA17512.jpg) (1353 x 2939) (314.6 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA17512.tif) (1353 x 2939) (4.0 MB)

Caption:


Context image

Dark streaks mark the steep slopes of this ridge located east of Mangala Valles. One theory about the creation of this feature is that downslope movement of a rock "clears" the brighter dust revealing the darker surface below.
