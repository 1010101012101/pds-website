# Flooded Place * Click the [image above](/press_releases/medium/PIA08xxx/PIA08643_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA08643.jpg) (2048 x 4696) (2.3 MB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA08643.tif) (2048 x 4696) (9.6 MB)

Caption:

26 July 2006
This Mars Global Surveyor (MGS) Mars Orbiter Camera (MOC) image shows a portion of a flood-carved canyon within the larger Kasei Valles system on Mars. This canyon is the result of the very last flood event that poured through the Kasei valleys, long ago.

Image width : ~3 km (~1.9 mi)
