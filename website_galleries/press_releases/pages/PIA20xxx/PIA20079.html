# Capri Mensa - False Color * Click the [image above](/press_releases/medium/PIA20xxx/PIA20079_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA20079.jpg) (709 x 1665) (90.9 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA20079.tif) (709 x 1665) (2.5 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of Capri Mensa. Elevations drop from the bottom of the image to the top. The materials that are blueish in the image are moving down slope and forming dunes.
