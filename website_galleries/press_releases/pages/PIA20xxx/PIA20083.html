# Escorial Crater- False Color * Click the [image above](/press_releases/medium/PIA20xxx/PIA20083_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA20083.jpg) (1357 x 1951) (121.0 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA20083.tif) (1357 x 1951) (3.0 MB)

Caption:


Context image

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of the mesa that Escorial Crater is located on. This mesa has steep margins, part of which is shown in this false color image.
