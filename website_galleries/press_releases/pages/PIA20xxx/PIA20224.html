# Hebes Mensa - False Color * Click the [image above](/press_releases/medium/PIA20xxx/PIA20224_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA20224.jpg) (691 x 1538) (93.2 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA20224.tif) (691 x 1538) (2.1 MB)

Caption:


Context image

The THEMIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows part of Hebes Mensa, the central higland feature in Hebes Chasma.
