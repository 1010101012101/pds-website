# Terra Cimmeria - False Color * Click the [image above](/press_releases/medium/PIA22xxx/PIA22671_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA22671.jpg) (607 x 2726) (120.3 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA22671.tif) (607 x 2726) (2.3 MB)

Caption:

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows several craters and plains of Terra Cimmeria.
