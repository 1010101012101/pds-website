# Proctor Crater Dunes - False Color * Click the [image above](/press_releases/medium/PIA22xxx/PIA22670_med.jpg) for a larger view * [Full-Res JPEG](https://photojournal.jpl.nasa.gov/jpeg/PIA22670.jpg) (615 x 2732) (99.5 kB) * [Full-Res TIFF](https://photojournal.jpl.nasa.gov/tiff/PIA22670.tif) (615 x 2732) (2.1 MB)

Caption:

The THEMIS VIS camera contains 5 filters. The data from different filters can be combined in multiple ways to create a false color image. These false color images may reveal subtle variations of the surface not easily identified in a single band image. Today's false color image shows the floor of Proctor Crater, including the large dune field. Basaltic sands are typically blue in false color images. Proctor Crater is located in Noachis Terra.
