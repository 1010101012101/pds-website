# Spacecraft-Based Ring Occultations ## Cassini 1/17/19 Version 2 of the RSS dataset [CORSS_8001](/link/volumes/CORSS_8xxx/CORSS_8001/) is available online. Version 2 includes reprocessed versions of the material contained in the earlier version, plus all subsequent RSS occultations obtained prior to the loss of the Cassini UltraStable Oscillator (USO) in late 2011. In addition, this version includes 'DLP' files for each observation which contain higher resolution calibrated but diffraction-limited optical depth and phase shift profiles of Saturn's rings, that is, calibrated profiles before reconstruction to remove diffraction effects. Version 1 contained only X band data files. Version 2 contains K, S, and X band data. Details of the data processing involved in the production of this data set are provided in MAROUFETAL1986. Versions 2 of the UVIS and VIMS datasets will extend the coverage of each dataset to all high quality occultations observed by the instrument over the course of the entire mission. For each dataset version 2 will be released as it completes the peer review process. We anticipate this to happen early in 2019. Version 1 of each of the datasets of Saturn ring occultation profiles from Cassini instruments (RSS, UVIS, VIMS) are available on line. For each dataset, version 1 contains occultations from the prime mission (first four years at Saturn). * **RSS version 2** \- Browse the data set on-line [CORSS_8001 version 2](/link/volumes/CORSS_8xxx/CORSS_8001/), or download the entire data set [CORSS_8001 tar.gz](/link/archives-volumes/CORSS_8xxx/CORSS_8001.tar.gz). * **RSS version 1** \- Browse the data set on-line [CORSS_8001 version 1](/link/volumes/CORSS_8xxx/CORSS_8001_v1/), or download the entire data set [CORSS_8001 version 1 tar.gz](/link/archives-volumes/CORSS_8xxx_v1/CORSS_8001.tar.gz). * **UVIS version 1** \- Browse the data set on-line [COUVIS_8001](/link/volumes/COUVIS_8xxx/COUVIS_8001/), or download the entire data set [COUVIS_8001 tar.gz](/link/archives-volumes/COUVIS_8xxx/COUVIS_8001.tar.gz). * **VIMS version 1** \- Browse the data set on-line [COVIMS_8001](/link/volumes/COVIMS_8xxx/COVIMS_8001/), or download the entire data set [COVIMS_8001 tar.gz](/link/archives-volumes/COVIMS_8xxx/COVIMS_8001.tar.gz). Details about each of the instruments are available from our [Voyager information pages](/voyager/). # Earth-Based Ring Occultations ## Saturn We have a single volume of Earth-based, Saturn stellar occultations. It contains six small data sets from the July 1989 occultation of the bright star 28 Sgr by Saturn's rings. Each data set contains observations from a single telescope. The telescopes are the European Southern Observatory's 1 meter telescope, the European Southern Observatory's 2.2 meter telescope, the IRTF, the Lick 1 meter telescope, the McDonald 2.7 meter telescope, and the Palomar 200 inch telescope. * Browse the data sets on-line [EBROCC_0001](/link/volumes/EBROCC_xxxx/EBROCC_0001/), or download the entire six data set volume [EBROCC_0001 tar.gz](/link/archives-volumes/EBROCC_xxxx/EBROCC_0001.tar.gz).
