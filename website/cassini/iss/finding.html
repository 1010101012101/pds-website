Search Options * **Use [OPUS](http://pds-rings-tools.seti.org/opus/) to search for specific Cassini ISS data products using a wide range of criteria including enhanced geometric metadata produced at the Rings node. The enhanced geometric metadata includes information appropriate to planet and satelllite searches as well as enhanced ring geometry.** * Search results include data files, tables of associated metadata, and browse thumbnails for each image. User selected data files can be bundled as tar.gz files for easy downloading. * The [Planetary Atlas.](http://pdsimg.jpl.nasa.gov/search/search.html#QuickSearch) at the [PDS Imaging Node](http://pds-imaging.jpl.nasa.gov/) has been significantly upgraded and supports a wide range of criteria for Cassini ISS data searches. * The [Access ISS Data](access.html) tab, provides links for each data volume, including links to the volume, to tar.gz files of the volume, and links to directories containing 200x200 "thumbnail" versions of the images on the volume. * The [ISS Calibration](calibration.html) tab, provides links for each calibration volume, including links to the volume and to zip files of the volume. * You can use the VOLUME_ID coupled with the index files provided on the ISS volumes to help you find specific data (see below). ![\[SPACER\]](../css/spacer.gif) ### 2\. Interpreting the VOLUME_ID ![\[SPACER\]](../css/spacer.gif) The volume ID takes the form **COISS_Nxxx**, where: **COISS** is the PDS abbreviation for Cassini Orbiter, Imaging Science Subsytem **N** is one of the following the mission phase identifiers: * **0** \- Calibration data * **1** \- Launch through Saturn approach * **2** \- Saturn tour **xxx** is the sequential volume number, begining with 001 for each of the three data sets. ![\[SPACER\]](../css/spacer.gif) ### 3\. The volume INDEX files ![\[SPACER\]](../css/spacer.gif) Each volume has an [index](/vol/COISS_2xxx/COISS_2001/index/) subdirectory containing three files: * [indxinfo.txt](indxinfo.txt): summarizes the structure and contents of the other two files in the INDEX subdirectory. **Users are strongly encouraged to read an indxinfo.txt before working with the INDEX files.** * [index.tab](COISS_2001_index_tab.txt): an ASCII table file with one row for each image on the volume. For Cassini ISS, each index table contains 118 columns. Each row contains keyword values for one image. This example is from disk COISS_2001. **NOTE** that these files are large, typically, 12 Mbytes. * [index.lbl](index_lbl.txt): the detached PDS label file for the indx.tab file. This is a text file giving names and definitions of all the columns in the index. ![\[SPACER\]](../css/spacer.gif) ### 4\. Useful Keywords for Ring searches ![\[SPACER\]](../css/spacer.gif) Columns in the index file that may be particularly useful to ring scientists include those for: RINGS_FLAG, TARGET_NAME, TARGET_DISTANCE, MAXIMUM_RING_RADIUS, MINIMUM_RING_RADIUS, RING_CENTER_LATITUDE, RING_CENTER_LONGITUDE, RING_EMISSION_ANGLE, RING_INCIDENCE_ANGLE, and OBSERVATION_ID The **OBSERVATION_ID** is a string constructed from the instrument ID, the orbit number, the target code, a description of the observation, and instance number. For example, VIMS_000RI_RINGMOS103_PRIME is some of the data taken by VIMS during orbit 000 to produce a radial mosaic of the rings. Searching the index file for the substring RI_ will identify general ring observations.
