Directory Structure The following directory structure is applicable to each Cassini RPWS Standard Product Archive Volume. In the cases where all data product types are not acquired during a given time interval covered by a volume, the relevant directories will simply be empty except for the INFO.TXT file. The directories are discussed in detail in the text of this document, except for EXTRAS, which contains the HTML interface accessible through AAREADME.HTM.
