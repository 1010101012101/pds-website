The data include ring radial profiles at 1 and 10 km resolution from more than 60 high quality VIMS occultations. The data set is still being compiled and has not undergone peer review. ** Since the data has not completed peer review, users should exercise caution when using the data.** * Users may browse the peer review version of the occultation data set [COVIMS_8001](/vol/COVIMS_8xxx_peer_review/COVIMS_8001/), or download the entire data set [COVIMS_8001 tar.gz](/archives/COVIMS_8xxx_peer_review/COVIMS_8001.tar.gz). * Please [ contact us](/cgi-bin/comment-form.pl?) if you encounter difficulties with the data, identify errors in the data set, or have suggestions for improvement of the data set. ### Getting the Data You Want ![\[SPACER\]](../css/spacer.gif) The table below contains temporal coverage information for, and links to, all of the available VIMS data volumes from cruise, Jupiter, and Saturn. * **Review** a volume on-line by clicking the volume ID. * **Download** an entire volume, by right clicking on that volume's "Bundled" link in the second table. The bundled volumes are provided in .tar.gz format. For information on opening files of this type, click [here](/targzip.htm). * **PREVIEW** will take you to a directory containing sub-directories of preview images for each VIMS cube in the volume. The preview images contain a wealth of information. The key for the preview images is in [COVIMS_previews.txt](COVIMS_previews.txt). ![\[SPACER\]](../css/spacer.gif) ### Available Volumes * To determine which volumes meet the publicly available criteria for a specific ROSES Announcement please see the appropriate DAP page under our [Proposal Support](/roses) pages.
