Documentation for the CIRS Re-formatted Data ![\[SPACER\]](/css/spacer.gif) There are three key documents specific to the re-formatted data set: * [AAREADME.TXT](reform_AAREADME.TXT) file. A copy of this file is found in the root directory of every re-formatted volume. * [TUTORIAL.TXT](TUTORIAL.TXT) file. A copy of this file is found in the DOCUMENT directory of every re-formatted volume. * [DATASET.CAT](reform_DATASET.txt) file. A copy of this file is found in the CATALOG directory of every re-formatted volume. ![\[SPACER\]](/css/spacer.gif) ### 2\. Additional Documentation ![\[SPACER\]](/css/spacer.gif) * A number of additional documents are included in the [DOCUMENT](/volumes/COCIRS_5403/DOCUMENT/) subdirectory on each volume. Those files are summarized in the [DOCINFO.TXT](reform_DOCINFO.TXT) file. A copy of this file is included in the DOCUMENT subdirectory of every re-formatted volume. * The [CATALOG](/volumes/COCIRS_5403/CATALOG/) subdirectory on each volume contains a wealth of information about the CIRS instrument, team and data set. See, in particular: * [COCIRS_xxxx/CATALOG/INST.CAT](INST.txt): Description of the CIRS instrument. * [COCIRS_xxxx/CATALOG/INSTHOST.CAT](INSTHOST.txt): Description of the Cassini spacecraft. * [COCIRS_xxxx/CATALOG/MISSION.CAT](MISSION.txt): Description of the Cassini mission. * A file, [ERRATA.TXT](reform_ERRATA.TXT), in the root directory of each volume documents any known anomalies or deviations from PDS standards contained in that volume. * Additional descriptions of the instrument can be found at the [ Cassini CIRS web page maintained at JPL.](//saturn.jpl.nasa.gov/spacecraft/inst-cassini-cirs-details.cfm) ![\[SPACER\]](/css/spacer.gif) ### 3\.
