Two instruments on the Galileo Spacecraft acquired data on Jupiter's Ring System. * The Solid State Imager (SSI) obtained high-resolution images at visual and infrared wavelengths. * The Near-Infrared Mapping Spectrometer (NIMS) obtained multi-spectral images at wavelengths from 0.7 to 5.2 microns. ## Finding Galileo Data * **Use [OPUS](//tools.pds-rings.seti.org/opus) to search for specific Galileo SSI data products using a wide range of criteria. ** While the [PDS Imaging Node](//pds-imaging.jpl.nasa.gov/) is the primary repository for all Galileo SSI and NIMS data, the Ring-Moon Systems Node maintains subsets of the data obtained by these two instruments, including all of the Jovian Ring observatons. Pages of browse images of the Galileo ring observations are provided below. ### Solid State Imager * [ C3 Encounter ](ssi_c3_data.html) (13 images; released December 19, 1997) * [ G8 Encounter ](ssi_g8_data.html) (2 Adrastea images; released June 25, 1998) * [ C10 Encounter ](ssi_c10_data.html) (10 ring images + 16 Europa images; released November 6, 1998) * [ E11 Encounter ](ssi_e11_data.html) (6 images; released December 16, 1998) * [ E17 Encounter ](ssi_e17_data.html) (11 images) * [ G28 Encounter ](ssi_g28_data.html) (5 images) * [ G29 Encounter ](ssi_g29_data.html) (6 images) * [ I32 Encounter ](ssi_i32_data.html) (3 images) * [ Viewing Geometry Table ](ssi_table.html) ### Near-Infrared Mapping Spectrometer * [ C3 Encounter ](nims_data/c3) (5 spectral image cubes) * [ E4 Encounter ](nims_data/e4) (1 spectral image cube) ## Links outside the Ring-Moon Systems Node Access all Galileo SSI and NIMS data at the PDS Imaging Node by using the Imaging Node's updated [ Planetary Image Atlas](//pds- imaging.jpl.nasa.gov/search/search.html#QuickSearch), or by browsing data volumes from the [ Galileo page](//pds- imaging.jpl.nasa.gov/Missions/Galileo_mission.html) at the Imaging Node.
