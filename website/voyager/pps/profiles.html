Standard PPS Ring Profiles

This page provides access to ASCII table files containing information about the rings at uniformly-spaced radial samples (0.1, 0.2, 0.5, 1, 2, 5, 10, 20, 50 km). The columns tabulated are:

  • Radius in km
  • Raw PPS counts
  • Standard deviation in raw PPS counts
  • Normal optical depth
  • One standard deviation lower limit on optical depth
  • One standard deviation upper limit on optical depth

Click on the data file name (*.TAB) to retrieve the data. Click on the corresponding label file (*.LBL) to retrieve the PDS label, which describes the data file contents in detail.

Note that all of these files are located within subdirectories of the EASYDATA directory on PDS volume VG_2801.

  • Saturn
  • Uranus
    • Ring 6
    • Ring 5
    • Ring 4
    • Ring alpha
    • Ring beta
    • Ring eta
    • Ring gamma
    • Ring delta
    • Ring lambda
    • Ring epsilon
    • Full Ring System (assuming equatorial geometry)
  • Neptune
