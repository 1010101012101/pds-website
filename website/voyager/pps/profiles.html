The columns tabulated are: * Radius in km * Raw PPS counts * Standard deviation in raw PPS counts * Normal optical depth * One standard deviation lower limit on optical depth * One standard deviation upper limit on optical depth Click on the data file name (*.TAB) to retrieve the data. Click on the corresponding label file (*.LBL) to retrieve the PDS label, which describes the data file contents in detail. Note that all of these files are located within subdirectories of the [EASYDATA](/vol/VG_28xx/VG_2801/EASYDATA/) directory on PDS volume [VG_2801](/vol/VG_28xx/VG_2801/). If none of these files meet your needs, use the [PPS Resampler](resampler.html) to generate a table to your specifications. * Saturn * Uranus * Ring 6 * Ring 5 * Ring 4 * Ring alpha * Ring beta * Ring
