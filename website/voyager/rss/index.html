--- layout: base layout_style: default tabs: voyager_rss --- ## Radio Science Subsystem The RSS instrument acquired profiles of the ring systems of Saturn and Uranus; no rings were detected at Jupiter or Neptune. PDS volume **VG_2803** contains the complete Voyager RSS ring occultation data set. It has completed peer review and is currently in lien resolution at the Ring-Moon Systems Node. * Browse the [on-line volume](/link/volumes/VG_28xx_peer_review/VG_2803) * Download the entire volume as a [bundled volume ](/link/archives-volumes/VG_28xx_peer_review/VG_2803.tar.gz) The bundled volume is in .tar.gz format. For information on opening files of this type, click [here](/help/targz.html). For an overview of the data set, see [AAREADME.TXT](aareadme.html) and [TUTORIAL.TXT](tutorial.html). For more information, consult the CATALOG files: * [DATASET.CAT](dataset.html): RSS data set information. * [DSCOLL.CAT](dscoll.html): Information about the data set collection, comprising the PPS, UVS and RSS ring occultation experiments. * [VG1SINST.CAT](vg1sinst.html): RSS instrument description for Voyager 1 at Saturn. * [VG2UINST.CAT](vg2uinst.html): RSS instrument description for Voyager 2 at Uranus. * [VG1HOST.CAT](vg1host.html): Voyager 1 spacecraft description. * [VG2HOST.CAT](vg2host.html): Voyager 2 spacecraft description. * [MISSION.CAT](mission.html): Description of the Voyager mission overall. * [PERSON.CAT](person.html): Contact information for the RSS team members and the individuals involved in the production of this data set. * [REF.CAT](ref.html): Bibliographic references. * [VOLDESC.CAT](voldesc.html): Brief description of the archive volume.
