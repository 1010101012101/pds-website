    Data Set Collection Overview
    ============================
      This is a collection of ring occultation data sets obtained by
      the Voyager spacecraft at Saturn, Uranus, and Neptune. It
      includes stellar occultation data from the Photopolarimeter
      (PPS) and Ultraviolet Spectrometer (UVS). It also includes
      radio occultation data from the Radio Science Subsystem (RSS),
      in a processed format in which diffraction effects have been
      removed."

  DATA_SET_COLLECTION_USAGE_DESC = "

    Data Set Collection Usage Overview
    ==================================
      The intent of this data set collection is to provide all three
      ring occultation data sets from Voyager in a common format that
      is relatively easy to use. Edited data, calibration models and
      geometry files are provided in standardized formats and software
      tools are included so that all three data sets can be
      manipulated in similar ways. This will greatly simplify the
      comparative study of the data. It is hoped that this will
      establish a reference standard so that future ring occultation
      data sets are archived in a similar manner."
