--- layout: base layout_style: default tabs: voyager_iss --- ## Jupiter Ring Images We provide direct access to all of the Jovian ring images as part of our support for [Galileo](/galileo/index.html) image analysis. The following links provide thumbnails and viewing geometry information for the indicated subsets of the Voyager Jovian Ring images. * [Voyager 1 ](/galileo/vgr1_data.html)(1 image) * [ Voyager 2 inbound ](/galileo/vgr2_lophase_data.html)(12 images) * [ Voyager 2 outbound ](/galileo/vgr2_hiphase_data.html) (12 images). All of these images are summarized in this [ viewing geometry table](/galileo/vgr_table.html). ## Derived Saturn Ring Profiles [VG_2810](/link/volumes/VG_28xx/VG_2810), contains radial profiles from the finest- resolution Voyager images. Profiles of both the lit and dark sides are included at various resolutions of order 5 km. We generally recommend using the data files sampled at 2-km resolution: * Dark side mosaic: [ IS1_P0001_V01_KM002.TAB](/link/volumes/VG_28xx/VG_2810/DATA/IS1_P0001_V01_KM002.TAB) and [.LBL](/link/volumes/VG_28xx/VG_2810/DATA/IS1_P0001_V01_KM002.LBL) * Lit side mosaic: [ IS2_P0001_V01_KM002.TAB](/link/volumes/VG_28xx/VG_2810/DATA/IS2_P0001_V01_KM002.TAB) and [.LBL](/link/volumes/VG_28xx/VG_2810/DATA/IS2_P0001_V01_KM002.LBL) * Other resolutions are available in the same directory, [DATA](/link/volumes/VG_28xx/VG_2810/DATA/). You can also download the entire volume as a tar.gz file [here](/link/archives-volumes/VG_28xx/VG_2810.tar.gz). For details on the volume structure and file naming conventions, please review the volume's AAREADME.TXT file (especially section 4).
