--- layout: base layout_style: default tabs: voyager --- ## Overview SPICE kernels contain critical information about the geometry, pointing, and timing of NASA's spacecrafts. For more information about SPICE visit the home page of PDS's [Navigation and Ancillary Information Facility](//pds-naif.jpl.nasa.gov) (NAIF). ## The Pointing Information (C kernels) The Ring-Moon Systems Node has generated Voyager C kernels from mission SEDR files. These C kernels are available through the Pointing Tab: * [Pointing](ck/) \- C kernels and other pointing information for Voyager's instruments. ## Other kernels The primary source of SPICE data is NAIF's [anonymous FTP site](ftp://naif.jpl.nasa.gov/pub/naif/). See, in particular: * [/pub/naif/VOYAGER](ftp://naif.jpl.nasa.gov/pub/naif/VOYAGER/): Voyager-related files. * [/pub/naif/generic_kernels](ftp://naif.jpl.nasa.gov/pub/naif/generic_kernels/): Generic kernels, e.g., the latest planetary ephemerides, constants and leap seconds. * [/pub/naif/toolkit](ftp://naif.jpl.nasa.gov/pub/naif/toolkit/): The SPICE software toolkit, ready-made for most popular platforms. * [/pub/naif/toolkit_docs](ftp://naif.jpl.nasa.gov/pub/naif/toolkit_docs/): Users' guides and tutorials.
